Rule Format Guide

Required Format

ALLOW TCP 22 FROM 0.0.0.0/0

  • ALLOW permits traffic.
  • DENY blocks traffic.
  • TCP is the network protocol.
  • 22 is the port number.
  • FROM is required in each rule.
  • 0.0.0.0/0 means open to the public internet.

Common Ports

22 SSH 23 Telnet 80 HTTP 443 HTTPS 3306 MySQL 5432 PostgreSQL 3389 RDP 6379 Redis 8080 Alt HTTP
